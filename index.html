¿Qué es CSS?

CSS describe cómo se deben mostrar los elementos HTML en la pantalla.

/* HTML es la estructura (el esqueleto).
   CSS es el estilo (la ropa y el maquillaje). */

/* Sin CSS, la web sería solo texto plano 
   en blanco y negro. */
2 / 20

Anatomía de una Regla CSS

Una regla CSS se compone de un Selector y un Bloque de Declaración.

selector {
  propiedad: valor;
}

/* Ejemplo real */
h1 {
  color: blue;
  font-size: 24px;
}
3 / 20

1. El Selector

Indica al navegador a qué elemento HTML queremos aplicar los estilos.

/* Selector de Etiqueta */
p {
  color: red; /* Aplica a TODOS los párrafos <p> */
}

/* Selector Universal */
* {
  margin: 0; /* Aplica a TODO */
}
4 / 20

Selector de Clase (.)

Selecciona elementos con un atributo class específico. Se usa un punto.

/* En CSS */
.boton-alerta {
  background-color: orange;
  font-weight: bold;
}

/* En HTML:
   <button class="boton-alerta">Cuidado</button> */
5 / 20

Selector de ID (#)

Selecciona un elemento único. Se usa una almohadilla #.

/* En CSS */
#cabecera-principal {
  height: 100px;
}

/* En HTML:
   <header id="cabecera-principal">...</header> */
6 / 20

Agrupación de Selectores

Si varios elementos comparten el mismo estilo, sepáralos por comas.

/* Aplica a h1, h2 Y párrafos */
h1, h2, p {
  font-family: "Arial", sans-serif;
  color: #333;
}
7 / 20

2. El Bloque de Declaración

Va entre llaves { }. ¡No olvides el punto y coma ;!

body {
  /* Declaración 1 */
  background-color: white; 

  /* Declaración 2 */
  color: black; 
}
8 / 20

Propiedades y Valores

La propiedad es qué cambias, el valor es cómo lo cambias.

p {
  /* Propiedad: color | Valor: red */
  color: red;

  /* Propiedad: text-align | Valor: center */
  text-align: center;
}

Comentarios en CSS

Ignorados por el navegador, útiles para humanos.

/* Esto es un comentario de una línea */

p {
  color: blue; /* Comentario al final */
}

/* 
   Este es un comentario
   de múltiples líneas
*/

Colores en CSS

Tres formas principales de definir colores.

div {
  /* Keywords */
  color: red; 

  /* Hexadecimal (Común) */
  background-color: #ff0000; 

  /* RGB */
  border-color: rgb(255, 0, 0);
}
11 / 20

El Modelo de Caja (Box Model)

¡Concepto Crítico! Todo elemento HTML es una caja.

.caja {
  width: 300px;   /* Contenido */
  padding: 20px;  /* Relleno interno */
  border: 5px solid black; /* Borde */
  margin: 50px;   /* Espacio externo */
}
12 / 20

Padding vs Margin

Diferencia clave: Espacio interno vs Espacio externo.

/* PADDING (Relleno) */
/* DENTRO de la caja. Toma el color de fondo. */
padding: 20px;

/* MARGIN (Margen) */
/* FUERA de la caja. Es invisible y empuja elementos. */
margin: 20px;

Propiedades de Texto

Control básico de tipografía.

p {
  font-family: "Helvetica", sans-serif;
  font-size: 16px;
  font-weight: bold;
  line-height: 1.5;
  text-decoration: none;
}
14 / 20

Unidades de Medida: Absolutas

Son fijas. Los pixeles (px) son los más comunes.

h1 {
  font-size: 32px;
}

/* Preciso, pero no siempre responsivo. */
15 / 20

Unidades de Medida: Relativas

Se adaptan. Esenciales para diseño moderno.

div {
  width: 50%; /* Mitad del padre */
}

p {
  font-size: 1.2rem; /* Basado en HTML root */
  margin-bottom: 2em; /* Basado en fuente actual */
}
16 / 20

La Cascada (Cascading)

El orden importa. La última regla leída gana.

p { color: red; }

/* Esta regla está después, así que "gana" */
p { color: blue; }

/* Resultado: Azul */
17 / 20

Especificidad (El peso)

ID > Clase > Etiqueta.

/* Pesa poco (Etiqueta) */
p { color: red; } 

/* Pesa más (Clase) - GANADOR */
.mi-texto { color: green; }

/* El texto será VERDE aunque p esté antes. */
18 / 20

¿Cómo añadir CSS a HTML?

Usa un archivo externo con <link>.

<!-- En el <head> de tu HTML -->

<link rel="stylesheet" href="estilos.css">

<!-- Mantiene el código limpio y reutilizable. -->
