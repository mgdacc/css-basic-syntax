¿Qué es CSS?

CSS (Hojas de Estilo en Cascada) describe cómo se deben mostrar los elementos HTML en la pantalla o papel.

/* HTML es la estructura (el esqueleto).
   CSS es el estilo (la ropa y el maquillaje). */

/* Sin CSS, la web sería solo texto plano en blanco y negro. */
2 / 20

Anatomía de una Regla CSS

Una regla CSS se compone de un Selector y un Bloque de Declaración.

selector {
  propiedad: valor;
}

/* Ejemplo real */
h1 {
  color: blue;
  font-size: 24px;
}
3 / 20

1. El Selector

Indica al navegador a qué elemento HTML queremos aplicar los estilos. Existen varios tipos.

/* Selector de Etiqueta */
p {
  color: red; /* Aplica a TODOS los párrafos <p> */
}

/* Selector Universal */
* {
  margin: 0; /* Aplica a TODOS los elementos de la página */
}
4 / 20

Selector de Clase (.)

Es el más flexible y usado. Selecciona elementos que tengan un atributo class específico. Se usa un punto ..

/* En CSS */
.boton-alerta {
  background-color: orange;
  font-weight: bold;
}

/* En HTML coincidiría con:
   <button class="boton-alerta">Cuidado</button> */
5 / 20

Selector de ID (#)

Selecciona un elemento único. Un ID no debe repetirse en la misma página. Se usa una almohadilla #.

/* En CSS */
#cabecera-principal {
  height: 100px;
}

/* En HTML coincidiría con:
   <header id="cabecera-principal">...</header> */
6 / 20

Agrupación de Selectores

Si varios elementos comparten el mismo estilo, puedes separarlos por comas para ahorrar código.

/* En lugar de escribir reglas separadas... */
h1, h2, p {
  font-family: "Arial", sans-serif;
  color: #333;
}

/* La regla se aplica a los h1, a los h2 Y a los párrafos. */
7 / 20

2. El Bloque de Declaración

Va entre llaves { }. Contiene una o más declaraciones separadas por punto y coma ;.

body {
  /* Declaración 1 */
  background-color: white; 

  /* Declaración 2 */
  color: black; 
}
/* ¡No olvides el punto y coma al final de cada línea! */
8 / 20

Propiedades y Valores

La propiedad es el aspecto que quieres cambiar. El valor es el ajuste específico que aplicas.

p {
  /* Propiedad: color | Valor: red */
  color: red;

  /* Propiedad: text-align | Valor: center */
  text-align: center;
}

Comentarios en CSS

Se usan para explicar el código y son ignorados por el navegador. A diferencia de HTML, en CSS solo existe un tipo de comentario de bloque.

/* Esto es un comentario de una línea */

p {
  color: blue; /* Este comentario está al final de la línea */
}

/* 
   Este es un comentario
   de múltiples líneas
*/
10 / 20

Colores en CSS

Hay tres formas principales de definir colores.

div {
  /* 1. Palabras clave (Keywords) */
  color: red; 

  /* 2. Hexadecimal (Muy común) */
  background-color: #ff0000; 

  /* 3. RGB (Red, Green, Blue) */
  border-color: rgb(255, 0, 0);
}

El Modelo de Caja (Box Model)

¡Concepto Crítico! En CSS, todo elemento HTML es una caja rectangular.

.caja {
  width: 300px;   /* Ancho del contenido */
  padding: 20px;  /* Espacio INTERNO (relleno) */
  border: 5px solid black; /* El borde */
  margin: 50px;   /* Espacio EXTERNO (separación) */
}
12 / 20

Padding vs Margin

La confusión más común al empezar.

/* PADDING (Relleno) */
/* Es espacio DENTRO de la caja. 
   Si la caja tiene color de fondo, el padding se pinta. */
padding: 20px;

/* MARGIN (Margen) */
/* Es espacio FUERA de la caja.
   Empuja a otros elementos lejos. Es invisible. */
margin: 20px;

Propiedades de Texto

Controlan la tipografía y legibilidad.

p {
  font-family: "Helvetica", Arial, sans-serif; /* Tipo de letra */
  font-size: 16px;      /* Tamaño */
  font-weight: bold;    /* Grosor (negrita) */
  line-height: 1.5;     /* Altura de línea (espaciado vertical) */
  text-decoration: none; /* Quitar subrayado a enlaces */
}
14 / 20

Unidades de Medida: Absolutas

Son fijas y no cambian según el tamaño de pantalla o configuración.

h1 {
  font-size: 32px; /* Pixeles */
}

/* Los pixeles son precisos, pero no siempre
   son la mejor opción para webs responsivas
   o accesibles. */
15 / 20

Unidades de Medida: Relativas

Se adaptan en relación a otros valores. Son esenciales para el diseño moderno.

div {
  width: 50%; /* Ocupa la mitad de su padre */
}

p {
  font-size: 1.2rem; /* 1.2 veces el tamaño base del HTML */
  margin-bottom: 2em; /* 2 veces el tamaño de fuente actual */
}
16 / 20

La Cascada (Cascading)

Significa que el orden importa. Si hay dos reglas iguales, gana la última.

p {
  color: red;
}

/* Esta regla está después, así que "gana" */
p {
  color: blue;
}

/* Resultado: Los párrafos serán AZULES */
17 / 20

Especificidad (El peso)

No todos los selectores valen lo mismo. Un ID pesa más que una Clase, y una Clase más que una Etiqueta.

/* Pesa poco (Etiqueta) */
p { color: red; } 

/* Pesa más (Clase) - GANADOR */
.mi-texto { color: green; }

/* Aunque 'p' esté después, la clase es más específica
   y el texto será VERDE. */
18 / 20

¿Cómo añadir CSS a HTML?

La mejor forma es usar un archivo externo (.css).

<!-- En el <head> de tu HTML -->

<link rel="stylesheet" href="estilos.css">

<!-- Esto mantiene el HTML limpio y permite
     usar los mismos estilos en varias páginas. -->
